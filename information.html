Information About This Portfolio

About This Portfolio

Welcome to my personal portfolio website! This portfolio was created to showcase my skills, hobbies, achievements, and personal interests. Through this website, I aim to introduce myself and share some of the exciting projects I’ve been working on. It’s a place where you can get to know me, my work, and my journey in a more fun and interactive way!

How I Built This Portfolio

This website was developed by me using HTML, CSS, and a little bit of JavaScript. I’ve always wanted to build something that reflects who I am and what I can do. It was important to me that my portfolio felt like an online extension of my personality, so I made it simple, clean, and easy to navigate.

Technologies Used:

  • HTML: Used for creating the structure of the website, so visitors can easily access the information they need.
  • CSS: Applied to style the website and give it a fresh, modern, and colorful design that reflects my fun side!
  • JavaScript: While not extensively used yet, I plan on incorporating interactive elements in the future, such as cool animations and dynamic content!

Where I Built This Portfolio

This portfolio was built using the awesome platforms GitHub and Glitch, which are great for developers like me to share and collaborate on projects.

GitHub

GitHub is where I store the code for this website. It allows me to keep track of all the changes I make and even collaborate with others if I want to. I also use GitHub Pages to make the website live and accessible to everyone. It's my go-to tool for version control!

Glitch

Glitch helped me create a live prototype of this website. It’s great for quick testing, and I love how easy it is to use. In just a few clicks, you can deploy your code and have it running online. It’s a fantastic tool for beginners and experienced developers alike.

Fun Facts About This Portfolio

  • This website is fully responsive, which means it looks great on both desktop and mobile devices!
  • The design was inspired by my love for simplicity and functionality. I wanted it to be easy to navigate while still reflecting my personality.
  • While coding this portfolio, I learned some new CSS tricks that I'm excited to share in future projects!
  • One of my goals for this portfolio is to update it regularly with new projects, competitions, and experiences as I grow and develop as a developer and person!

Conclusion

Thank you for visiting my portfolio! I hope you enjoyed exploring my work and learning more about me. Feel free to get in touch if you'd like to collaborate or ask me anything. I am always open to new opportunities and challenges!
